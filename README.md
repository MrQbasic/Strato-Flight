# Strato-Flight
Data logging and transmission platform for high-altitude balloons.

Strato-Flight aims to replace a [commercial data logging module](https://www.stratoflights.com/shop/datenlogger/) from Stratoflights and extend its functionality with live data transmission via LoRa.

Unlike the commercial platform, which is closed-source and cannot be expanded or modified, Strato-Flight is fully customizable and designed for further development.

## Hardware
- [Heltec Wireless Tracker](https://heltec.org/project/wireless-tracker/)
- SD card reader
- I2C sensors

## Features
- GPS (position and time)
- Data and Error logging to SD card
- Live data transmission via LoRa
- Menu system on integrated Display
